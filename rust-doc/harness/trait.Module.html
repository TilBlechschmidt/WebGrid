Core run loop of the module

When the function returns Some(_) the death of the returned Heart is awaited before calling the shutdown hook. Useful for situations where you dispatch background jobs in the run loop but want to hand-off the program lifecycle management.

Returning None results in the program entering a shutdown state and calling the pre_shutdown hook.
